# Numbered for procedures, bullets for options, prose for reasoning

The list shape signals the content's nature. Numbered steps mean sequential procedure; bullets mean unordered options; prose means reasoning.

## How to apply

- A procedure with steps that must run in order: numbered list.
- A set of parallel options where order does not matter: bulleted list.
- A logical argument or explanation: prose paragraphs, not a list.

## Why

A bulleted list of steps signals "these can run in any order" when they cannot — readers may follow the bullets out of sequence and break the workflow. A numbered list of options signals "step 1 must happen before step 2" when nothing requires that. Prose stuffed into bullets fragments arguments that should flow.
