A list property that classifies a document.

## Details

Each value is a label — a descriptive or identifying epithet attached to the document (`note`, `design`, `todo`). Special only by wiki convention, but a list property like any other. Identity, not state: tags say what a document is and stay fixed, unlike a state property such as `status`.
