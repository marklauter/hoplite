# Em-dash usage

Use the em dash character (`—`) for parenthetical breaks, definitions, and appositives. Not double hyphens.

## How to apply

- Parenthetical breaks: `the cache — set to 300s — expires fast` (em dash with spaces on both sides).
- Definitions: `the lede — one or two sentences that name the artifact's intent`.
- Appositives: `the reviewer, the agent that judges, runs after the writer` could be `the reviewer — the agent that judges — runs after the writer`.

## What to avoid

- Double hyphens (`--`) — those are typewriter substitutes for the em dash; use the character itself.
- Single hyphens with spaces (` - `) where parenthetical breaks are intended; that's a violation.
