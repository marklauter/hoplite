# Link-strip test

Read the page imagining every link is broken. Does the core message still come through? If not, more context is needed inline.

## How to apply

- Mentally remove all hyperlinks from the page.
- Read what remains.
- Ask: does the message survive? Or did the links carry the load?
- If they carried the load, the page is not self-contained. Inline enough context to recover.

## What this catches

- Pages that rely on terminology definitions in linked glossaries without restating the gist.
- Pages that defer the actual content to a "see the spec" link.
- Pages that name concepts without explaining them, trusting the link to do the explaining.
