# Hedge and filler grep

Search the page for each hedge and filler word in the ban list. Each hit is a candidate for removal.

## The ban list

- Hedges: *might*, *perhaps*, *could be*, *it's worth noting*, *might want to*.
- Filler: *basically*, *simply*, *just*, *actually*, *really*, *quite*, *very*, *in order to*.

Each hit is a candidate; some survive (a *might* that genuinely signals uncertainty, for example). Most do not.

## When to keep

- The hedge expresses real uncertainty, not stylistic softness ("the cache might be stale" when stale-ness is genuinely unknown).
- The filler is part of a quoted example or worked transformation.
- Otherwise, cut.
